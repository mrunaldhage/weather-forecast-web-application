# Weather Forecast Web Application 

A responsive weather forecast web application built using **HTML, CSS, and JavaScript** that displays real-time weather information using the **OpenWeather API**.

## Features
- City-based weather search
- Displays temperature, feels-like temperature, humidity, wind speed, clouds, pressure
- Sunrise & sunset timings based on local timezone
- Celsius / Fahrenheit temperature conversion
- Responsive UI design

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## How to Use
1. Enter a city name in the search bar
2. Click the search icon
3. View real-time weather details instantly

## Setup Instructions
1. Clone the repository
2. Open `script.js`
3. Replace `YOUR_OPENWEATHER_API_KEY` with your own OpenWeather API key
4. Open `index.html` in a browser
